El Proceso con Maxive

Si bien es genial trabajar con una máquina virtual fácilmente, mucha gente no quiere editar archivos usando editores basados ​​en terminales a través de SSH. Afortunadamente con Maxive no es necesario. Al usar carpetas sincronizadas , Maxive sincronizará automáticamente sus archivos a y desde la máquina invitada.

1.

Define la Infraestructura

Crea una Carpeta Principal con Maxive y configura tu primer archivo de configuración Maxivefile.

2.

Aumenta tu productividad

Explora en la Nube de Maxive y busca máquinas virtuales preconfiguradas para un despliegue mas úrgente.

3.

Ejecuta una Caja Maxive

Al ejecutar maxive, las Máquinas Virtuales se crean a partir de Cajas o Boxes configurados con el Maxivefile, este archivo puede versionarse y utilizar git para controlar sus cambios.

4.

Aprende y Evoluciona

La Infraestructura es portable como código, puede adémas versionarse y evolucionar, luego reaprovisionarse cuando cambiamos su configuración para desplegarse con mayor eficiencia
